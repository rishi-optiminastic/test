# Integration

### Overview

**What are Integrations?**

Integrations are automated connections between CarbonCut and external data sources (cloud providers, CDN services, on-premise servers, HR platforms, etc.) that enable real-time carbon emission tracking without manual data entry. Instead of uploading CSV files or manually inputting data, integrations pull emissions data automatically through APIs.

**What we're doing:**

* Connecting to third-party services using their APIs
* Authenticating securely with API keys, OAuth, or IAM roles
* Fetching usage/consumption data (compute hours, bandwidth, energy consumption)
* Converting that data into carbon emissions using emission factors
* Syncing regularly to keep emissions data up-to-date
